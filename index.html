CSS Selectors

Selector Example Meaning
Universal Selector * {} Targets all elements on the page
Type Selector h1, h2, h3 {} Matches element names
Class Selector .intro {} Targets elements with a specific class attribute
ID Selector #introduction {} ID selector styles the element with the specified id
Child Selector li > a {} Targets any <a>elements that are children of an <li> element
Descendant Selector p a {} Targets any <a> that sits inside a <p> element
Adjacent Sibling Selector h1 + p {} Targets the first <p> after any <h1> element
General Sibling Selector h1 ~ p {} Targets both <p> elements of <h1> if you there is two
